# On-Muhasebe-Otomasyonu
Bu proje, Antigravity ile oluşturulmuş açık kaynaklı bir ön muhasebe masaüstü uygulamasıdır. Mevcut sürümdeki hataları birlikte gidermek ve yapay zeka desteğiyle projeyi daha ileriye taşımak için topluluk desteği bekliyorum. Haydi, bu projeyi hep beraber ücretsiz ve profesyonel bir muhasebe çözümüne dönüştürelim!
